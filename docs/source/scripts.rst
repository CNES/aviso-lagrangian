Console scripts
===============

This section provides an overview of the Python programs included in this
project. These scripts are designed to process and analyze geophysical data,
offering functionalities such as computing FSLE/FTLE maps, converting velocity
units, and simulating particle drift. Each script is documented in detail to
guide users through its usage and configuration.
